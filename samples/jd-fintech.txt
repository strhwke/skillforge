Senior Full-Stack Engineer — Nimbus, a Series B fintech for SMB lending

About the role
We're hiring a senior full-stack engineer to own a vertical slice of our underwriting platform — from
the React dashboards loan officers use, through the Node.js / GraphQL API, into the PostgreSQL data
model that backs every credit decision. You will work directly with the founding engineers and ship to
production weekly.

Responsibilities
• Design, build, and maintain features end-to-end across React, TypeScript, Node.js, and PostgreSQL
• Architect new GraphQL schemas and Apollo resolvers that scale with our 6x YoY transaction growth
• Own performance: profile, instrument, and optimize hot paths in the API and DB layer
• Lead the system-design discussion for one major initiative each quarter (e.g. real-time risk pipeline)
• Mentor mid-level engineers, run technical design reviews, raise the engineering bar
• Own observability: dashboards, SLOs, and on-call rotation for your services on AWS (ECS + RDS)
• Partner with data science to productionize ML risk models behind a low-latency inference service

Requirements
• 5+ years shipping production full-stack systems with React + Node.js
• Deep PostgreSQL knowledge: indexing strategy, query plans, transactional correctness
• Strong system design ability — can defend trade-offs on consistency, caching, and queueing
• Hands-on AWS experience (ECS, RDS, IAM, CloudWatch); comfortable in Terraform or CDK
• Demonstrated GraphQL experience in production (Apollo Server preferred)
• Excellent written communication — RFCs, design docs, post-mortems

Nice to have
• Experience in regulated industries (lending, payments, banking)
• Kubernetes (EKS) operational experience
• Real-time systems (websockets, event-driven, Kafka)
• Python for ML model integration
